# Flappy Bird Deep Q-Network (DQN) Agent

A Reinforcement Learning project that trains an AI agent to play Flappy Bird using a Deep Q-Network (DQN) built with PyTorch and Gymnasium.

## 📁 Project Structure

* `agent.py` - Core RL Agent class handling hyperparameter loading and action selection.
* `dqn.py` - Deep Q-Network neural network architecture.
* `experience_replay.py` - Replay memory buffer for stabilizing training.
* `game_fillpy_bird.py` - The main training loop execution script.
* `parameters.yaml` - Configuration file containing hyperparameters (learning rate, discount factor, etc.).
* `requirements.txt` - Project dependence
